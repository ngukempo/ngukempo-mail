# NGU KEMPO メール

ドメインは `m.ngukempo.com` です。

[NGU KEMPO メールを開く](https://m.ngukempo.com)

---

## 対象ユーザーについて

利用できるのは以下の学生です。

- 在学中の学生（入部済みであること）
- 卒業生

**転科した場合の注意：**
在学中に転科した場合、IDは転科後の学籍番号に変わります。
元の学籍番号では利用できなくなりますのでご注意ください。

**卒業生のIDについて：**
卒業生のIDは、卒業時の学籍番号です。
2024年度（24生）以降の卒業生には、卒業時に利用案内を行う予定です。

---

## メールアドレスについて

メールアドレスの形式は以下のとおりです。

```
学籍番号@m.ngukempo.com
```

**例：** 学籍番号が `99X9999` の場合

```
99x9999@m.ngukempo.com
```

## サービス利用にあたっての注意事項

- **不審なメールに注意してください。**
  知らない送信元からのメールはウィルスが含まれている可能性があります。
  添付ファイルを開いたり、本文内のリンクにアクセスしたりしないでください。

- **フィッシング詐欺に注意してください。**
  リンク先で個人情報（住所・氏名など）を入力する際は、企業を装った偽サイトの可能性があります。
  十分に確認してから入力してください。

- **ID・パスワードは厳重に管理してください。**
  他人に教えたり、貸したりしないでください。

- **ルールとマナーを守ってください。**
  実生活と同様に、他人のプライバシーや著作権を侵害しないようにしてください。

- **システム不具合について：**
  サービス利用中に発生したシステム的な不具合には、十分に対応できない場合があります。
